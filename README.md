# GDPR Overview

**GDPR** (General Data Protection Regulation) â€“ EU data privacy law effective **May 2018**.

## Key Points
- **Purpose:** Gives individuals control over their personal data.
- **Rights:** Know what data is collected, how it's used, correct or delete data.
- **Example:** E-commerce site collects only necessary info for newsletters.

## Impact on Data Work
- Must allow individuals to access, manage, and delete their data.
- Data handling must comply with privacy principles.

## Consequences of Breach
- Large fines.
- Mandatory reporting to **ICO**.
- Possible legal action from affected individuals.
